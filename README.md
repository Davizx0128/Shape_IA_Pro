# Shape_IA_Pro
ShapeAI é um aplicativo fitness inteligente desenvolvido como PWA (Progressive Web App), que oferece: Treinos organizados por nível Receitas saudáveis com cálculo de calorias Controle diário de metas Monitoramento de progresso Funcionalidade offline Instalação direta no celular Uma solução completa para evolução física e hábitos saudáveis.
